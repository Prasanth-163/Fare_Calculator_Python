# Fare Calculator (Python)

## Overview

This project is a simple fare calculation system developed using Python. It calculates the ride cost based on distance, vehicle type, and time of the day.

## Features

* Calculates fare based on distance
* Supports multiple vehicle types (Economy, Premium, SUV)
* Accepts short inputs like E, P, S
* Applies surge pricing during peak hours (5 PM to 8 PM)
* Handles invalid vehicle input

## Technologies Used

* Python
* Basic concepts: functions, dictionary, conditions, user input

## How to Run

1. Open terminal in project folder
2. Run the program:
   python fare_calc.py

Sample Input

* Distance: 10 km
* Vehicle: Economy
* Hour: 18

Sample Output

Distance: 10 km
Vehicle: Economy
Final Fare: Rs 150.0
